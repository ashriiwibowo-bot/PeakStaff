# PeakStaff
PeakStaff adalah prototipe aplikasi berbasis web (Front-End) yang dirancang untuk menghubungkan pelaku UMKM Food &amp; Beverage (F&amp;B) dengan freelancer (pekerja lepas) di sekitar mereka. Platform ini memberikan solusi bagi UMKM yang membutuhkan tenaga kerja mendadak (dalam waktu kurang dari 3 jam) saat menghadapi jam sibuk (peak hours).
